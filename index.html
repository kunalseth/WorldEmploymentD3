World Development Indicators: Exploring Unemployment Rates Vs. Average Internet Users (per 100 people)

The below interactive visualization helps you explore the Unemployment Rates Vs. Average Internet Users (per 100 people) across Regions, Countries, Income Groups, Population for the 5 years from 2010 to 2014.

The multidimensional visual analysis using cross-filtered views works on an "AND" conditioning, for example, if you select a Region X and Country Y -- the data will show up only if that country falls in that region. Also, there is no particular order to select the filter, data can be sliced on any level.

The idea is to allow you to narrow down the comparisons based on any sliced view that you would like to view the information in.

This is particularly helpful when you want to view data for a given region across income groups.

Also, the color scheme drop-down allows you to select your preferred color encoding to derive the best insights.
